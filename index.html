This project

This application was made by 6 students studying a course in information visualization at KTH in Stockholm.

The goal has been to visualize data generated by Blocket Jobb. For this project, mapping data to geographical locations and visualizing changes over time has been in the main focus.

This application provides two modes of comparison:

  • In the first mode, you can compare the amount of job ads of different Swedish regions.
  • In the second mode, you can view regional changes in job ad distributions.

Filtering of the data can be done by setting a time span for comparison, as well as by filtering data by category of job.
